Arrays in PHP

What are Arrays?

Arrays are used to store multiple values in a single variable. In PHP, arrays can hold multiple values of different types, such as strings, integers, or other arrays. Arrays help organize data and make it easier to manage.

1. Indexed Arrays

Indexed arrays use numeric indices to access values. By default, PHP assigns indices starting from 0.

<?php
$fruits = array("Apple", "Banana", "Cherry");
echo $fruits[0]; // Outputs: Apple
echo $fruits[1]; // Outputs: Banana
echo $fruits[2]; // Outputs: Cherry
?>

Output: Apple Banana Cherry

2. Associative Arrays

Associative arrays use named keys instead of numeric indices. You can define your own keys, which are more descriptive.

<?php
$person = array("name" => "John", "age" => 30, "city" => "New York");
echo $person["name"]; // Outputs: John
echo $person["age"];  // Outputs: 30
echo $person["city"]; // Outputs: New York
?>

Output: John 30 New York

3. Multidimensional Arrays

Multidimensional arrays are arrays containing one or more arrays. These are useful for storing complex data, like a table of values.

<?php
$students = array(
    array("John", 20, "A"),
    array("Jane", 22, "B"),
    array("Doe", 21, "C")
);
echo $students[0][0]; // Outputs: John
echo $students[1][1]; // Outputs: 22
echo $students[2][2]; // Outputs: C
?>

Output: John 22 C

4. Looping through Arrays

To loop through arrays, you can use different loops such as foreach, for, or while. The foreach loop is especially useful for iterating over arrays.

<?php
$colors = array("Red", "Green", "Blue");

foreach ($colors as $color) {
    echo $color . "<br>";
}
?>

Output:
Red
Green
Blue

5. Array Functions

PHP provides many built-in functions for working with arrays. These functions can be used for sorting, merging, filtering, and more. Here are some examples:

Sorting an Array

<?php
$numbers = array(4, 2, 8, 6);
sort($numbers); // Sorts the array in ascending order
print_r($numbers); // Prints the sorted array
?>

Output:
Array ( [0] => 2 [1] => 4 [2] => 6 [3] => 8 )

Array Merging

<?php
$array1 = array("Apple", "Banana");
$array2 = array("Cherry", "Date");
$mergedArray = array_merge($array1, $array2); // Merges two arrays
print_r($mergedArray); // Prints the merged array
?>

Output:
Array ( [0] => Apple [1] => Banana [2] => Cherry [3] => Date )

Array Filter

<?php
$numbers = array(1, 2, 3, 4, 5, 6);
$evenNumbers = array_filter($numbers, function($num) {
    return $num % 2 == 0;
});
print_r($evenNumbers); // Filters even numbers
?>

Output:
Array ( [1] => 2 [3] => 4 [5] => 6 )

6. Adding and Removing Elements

You can add or remove elements from an array using functions like array_push(), array_pop(), array_shift(), and array_unshift().

Adding an Element

<?php
$fruits = array("Apple", "Banana");
array_push($fruits, "Cherry"); // Adds "Cherry" to the end
print_r($fruits);
?>

Output:
Array ( [0] => Apple [1] => Banana [2] => Cherry )

Removing an Element

<?php
array_pop($fruits); // Removes the last element
print_r($fruits);
?>

Output:
Array ( [0] => Apple [1] => Banana )
