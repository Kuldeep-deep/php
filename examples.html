Example 1: Simple Calculator

This is a basic example of a calculator that performs addition, subtraction, multiplication, and division.

<?php
// Simple Calculator Example
$num1 = 10;
$num2 = 5;

// Addition
$sum = $num1 + $num2;
echo "Sum: " . $sum . "<br>";

// Subtraction
$diff = $num1 - $num2;
echo "Difference: " . $diff . "<br>";

// Multiplication
$product = $num1 * $num2;
echo "Product: " . $product . "<br>";

// Division
$quotient = $num1 / $num2;
echo "Quotient: " . $quotient . "<br>";
?>

Explanation: This example demonstrates basic arithmetic operations (addition, subtraction, multiplication, and division) in PHP using variables.

Example 2: Checking Prime Numbers

This example checks if a number is prime.

<?php
// Check if a number is prime
function isPrime($num) {
    if ($num <= 1) {
        return false;
    }
    for ($i = 2; $i <= sqrt($num); $i++) {
        if ($num % $i == 0) {
            return false;
        }
    }
    return true;
}

$number = 7;
if (isPrime($number)) {
    echo $number . " is a prime number.";
} else {
    echo $number . " is not a prime number.";
}
?>

Explanation: The function isPrime() checks if a given number is prime by dividing it by all numbers up to the square root of the number.

Example 3: Fibonacci Sequence

This example generates the first 10 numbers in the Fibonacci sequence.

<?php
// Fibonacci Sequence
function fibonacci($n) {
    $sequence = [0, 1];
    for ($i = 2; $i < $n; $i++) {
        $sequence[] = $sequence[$i - 1] + $sequence[$i - 2];
    }
    return $sequence;
}

$numbers = fibonacci(10);
echo "First 10 Fibonacci numbers: " . implode(", ", $numbers);
?>

Explanation: The fibonacci() function generates a sequence of Fibonacci numbers by starting with 0 and 1 and summing the last two numbers to generate the next one.

Example 4: Simple Form Validation

This example demonstrates simple form validation that checks whether the user has entered a name.

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = $_POST['name'];
    if (empty($name)) {
        echo "Name is required.";
    } else {
        echo "Hello, " . htmlspecialchars($name) . "!";
    }
}
?>

Explanation: This form checks if the user has entered a name. If the field is empty, it displays a message saying "Name is required." Otherwise, it greets the user by their name.
