Control Structures in PHP

What are Control Structures?

Control structures are used to control the flow of a program based on different conditions and loops. PHP offers several types of control structures to handle conditional statements and loops.

1. If-Else Statements

The if statement executes code if a condition is true. The else statement executes code if the condition is false, and elseif allows for multiple conditions.

<?php
$age = 20;

if ($age >= 18) {
    echo "You are an adult.";
} elseif ($age >= 13) {
    echo "You are a teenager.";
} else {
    echo "You are a child.";
}
?>

Output: You are an adult.

2. Switch Statement

The switch statement is an alternative to if for checking multiple possible values of a single variable.

<?php
$day = "Wednesday";

switch ($day) {
    case "Monday":
        echo "Start of the work week!";
        break;
    case "Wednesday":
        echo "It's midweek!";
        break;
    case "Friday":
        echo "Almost the weekend!";
        break;
    default:
        echo "Just another day.";
}
?>

Output: It's midweek!

3. For Loop

The for loop is used to run a block of code a specific number of times, ideal when the iteration count is known in advance.

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Number: " . $i . "<br>";
}
?>

Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

4. While Loop

The while loop repeats code as long as a condition is true. It's useful when the number of iterations is not predetermined.

<?php
$count = 1;

while ($count <= 3) {
    echo "Count is: " . $count . "<br>";
    $count++;
}
?>

Output:
Count is: 1
Count is: 2
Count is: 3

5. Do-While Loop

The do-while loop is similar to while, but it guarantees at least one execution of the loop, as the condition is checked after the code block runs.

<?php
$count = 1;

do {
    echo "Count is: " . $count . "<br>";
    $count++;
} while ($count <= 3);
?>

Output:
Count is: 1
Count is: 2
Count is: 3

6. Foreach Loop

The foreach loop is specifically designed for looping through arrays. It runs a block of code for each element in an array.

<?php
$colors = array("red", "green", "blue");

foreach ($colors as $color) {
    echo "Color: " . $color . "<br>";
}
?>

Output:
Color: red
Color: green
Color: blue
