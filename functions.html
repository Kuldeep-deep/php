Functions in PHP

What are Functions?

Functions in PHP allow you to group code into reusable blocks. A function performs a specific task and can be called multiple times throughout your script. Functions help make your code modular, reusable, and easier to maintain.

1. Defining a Function

A function in PHP is defined using the function keyword, followed by the function name and parentheses. Inside the function, you can place the code that will be executed when the function is called.

<?php
function greet() {
    echo "Hello, world!";
}

greet(); // Calls the function
?>

Output: Hello, world!

2. Functions with Parameters

You can define functions with parameters, allowing you to pass values to them when they are called. This allows for more dynamic behavior in your functions.

<?php
function greet($name) {
    echo "Hello, " . $name . "!";
}

greet("Alice"); // Passes "Alice" as an argument
greet("Bob");   // Passes "Bob" as an argument
?>

Output:
Hello, Alice!
Hello, Bob!

3. Returning Values from Functions

A function can return a value to the caller using the return keyword. You can use this returned value for further processing or output.

<?php
function add($a, $b) {
    return $a + $b;
}

$result = add(5, 10); // Calls the function and stores the result in $result
echo "The sum is: " . $result; // Outputs the result
?>

Output: The sum is: 15

4. Default Arguments

Functions can have default argument values. If an argument is not provided when the function is called, the default value is used.

<?php
function greet($name = "Guest") {
    echo "Hello, " . $name . "!";
}

greet();      // Uses the default value "Guest"
greet("Alice"); // Uses the passed value "Alice"
?>

Output:
Hello, Guest!
Hello, Alice!

5. Variable Scope in Functions

Variables declared inside a function are local to that function and cannot be accessed outside of it. However, you can access global variables using the global keyword or by passing them as arguments.

<?php
$globalVar = "I am global";

function testScope() {
    global $globalVar; // Access global variable
    echo $globalVar; // Outputs "I am global"
}

testScope();
?>

Output: I am global

6. Built-in PHP Functions

PHP has many built-in functions for common tasks. You can use them without defining your own. For example, the strlen() function returns the length of a string.

<?php
$string = "Hello, PHP!";
echo "String length: " . strlen($string); // Outputs the length of the string
?>

Output: String length: 12

7. Anonymous Functions (Closures)

PHP also supports anonymous functions, which are functions without a name. These functions can be assigned to variables or passed as arguments.

<?php
$sum = function($a, $b) {
    return $a + $b;
};

echo $sum(3, 4); // Calls the anonymous function
?>

Output: 7
