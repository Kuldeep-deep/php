Variables & Data Types in PHP

What is a Variable?

In PHP, a variable is a storage location that holds a value, which can be changed during the script execution. All variables in PHP start with a $ symbol followed by the variable name.

Basic Syntax for Declaring a Variable:

<?php
$variable_name = value;
?>

For example, $name = "Alice"; declares a variable named $name with the value "Alice".

Data Types in PHP

PHP supports several data types that can be assigned to variables. The primary data types are:

  • String
  • Integer
  • Float (floating-point numbers)
  • Boolean
  • Array
  • Object
  • NULL

Examples of Data Types

1. String

A string is a sequence of characters, typically used for text.

<?php
$greeting = "Hello, World!";
echo $greeting;
?>

Output: Hello, World!

2. Integer

An integer is a non-decimal number. PHP supports both positive and negative integers.

<?php
$age = 30;
echo $age;
?>

Output: 30

3. Float

A float, or floating-point number, is a number with a decimal point.

<?php
$price = 9.99;
echo $price;
?>

Output: 9.99

4. Boolean

A boolean represents two possible states: true or false.

<?php
$is_logged_in = true;
echo $is_logged_in; // Outputs 1 for true
?>

Output: 1 (true is represented as 1, and false as an empty string)

5. Array

An array can hold multiple values in a single variable. Arrays can be indexed or associative.

<?php
$colors = array("red", "green", "blue");
echo $colors[0]; // Outputs "red"
?>

Output: red

6. NULL

NULL is a special data type that represents a variable with no value.

<?php
$x = null;
var_dump($x);
?>

Output: NULL

Using var_dump() to Check Data Types

The var_dump() function is useful for checking the type and value of a variable. It outputs the data type as well as the value of the variable.

<?php
$example = "PHP";
var_dump($example); // Outputs: string(3) "PHP"
?>

Output: string(3) "PHP"

Dynamic Typing in PHP

PHP is a loosely typed language, meaning you don’t need to declare a variable’s data type. The type is assigned based on the value. For example, $x = "Hello"; makes $x a string, but $x = 10; reassigns it as an integer.
